# 📊 Sales Performance Analysis Dashboard

## 📌 1. Overview

The **Sales Performance Analysis Dashboard** provides a visual summary
of sales data across regions, product categories, and time periods. It
enables users to quickly analyze revenue, orders, and product
performance through interactive filters.

### 🚀 Dashboard Capabilities

-   📈 Monitor overall sales performance
-   🌍 Compare revenue across regions
-   🛍️ Understand product category contribution
-   🗓️ Filter results by month and day of the week
-   🔎 Explore detailed sales breakdowns

------------------------------------------------------------------------

# 📊 2. Key Metrics Section

📍 **Location:** Top-Left Corner

This section displays high-level **Key Performance Indicators (KPIs)**
that summarize sales performance.

### 📌 Metrics Included

-   💰 **Total Revenue** -- Total sales revenue generated\
-   🧾 **Total Orders** -- Total number of orders placed\
-   📦 **Total Units Sold** -- Total quantity of products sold\
-   📊 **Average Order Value** -- Revenue generated per order\
-   💵 **Average Unit Price** -- Average selling price per unit

✅ **Note:** All metrics automatically update when filters are applied.

------------------------------------------------------------------------

# 🎛️ 3. Interactive Filter Buttons (Highlighted Controls)

## 📅 Month Name Buttons

📍 **Location:** Top Center

### 🗓️ Available Options

-   🔘 Jan\
-   🔘 Feb\
-   🔘 Mar

### ⚙️ Function

These buttons allow users to **filter dashboard data by month**.

### 💡 Example

-   Click **Jan** → Displays sales data only for January\
-   Click **Feb** → Displays sales data for February\
-   Click **Mar** → Displays sales data for March

📊 Helps users analyze **monthly sales trends and revenue patterns**.

------------------------------------------------------------------------

## 📆 Day of Week Buttons

📍 **Location:** Top Right

### 🗓️ Available Days

-   🔘 Monday\
-   🔘 Tuesday\
-   🔘 Wednesday\
-   🔘 Thursday\
-   🔘 Friday\
-   🔘 Saturday\
-   🔘 Sunday

### ⚙️ Function

Filters the dashboard based on the **day orders were placed**.

### 💡 Example Use

-   📊 Select **Saturday & Sunday** → Analyze weekend sales\
-   📊 Select **Monday--Friday** → Evaluate weekday performance

This helps identify **which days generate the most revenue or orders**.

------------------------------------------------------------------------

## 🌍 Region Filter

📍 **Location:** Top Center Table
