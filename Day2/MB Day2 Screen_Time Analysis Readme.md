# 📊 Educational Analytics: Impact of Screen Time on Student Performance

## 📌 Project Overview

This project analyzes the relationship between **student screen time and
academic performance** using Microsoft Excel. The goal is to understand
how screen usage impacts **test scores, study habits, and
extracurricular activities**.

Using **data cleaning, pivot tables, and visualizations**, the analysis
provides actionable insights that help educators understand student
lifestyle patterns and their effect on academic outcomes.

------------------------------------------------------------------------

## 🎯 Project Objectives

-   📈 Analyze how screen time affects student test scores\
-   📚 Evaluate the relationship between study hours and academic
    performance\
-   ⚽ Understand how extracurricular activities vary with screen usage\
-   👩‍🎓 Identify age-wise performance trends\
-   📊 Create an interactive dashboard to visualize insights

------------------------------------------------------------------------

## 🗂 Dataset Information

  Column Name              Description
  ------------------------ ------------------------------------------
  Student_ID-               Unique identifier for each student
  \Age -                     Age of the student
  \Study_Hours  -            Average hours spent studying per day
  \Screen_Time  -            Average daily screen time (hours)
  \Test_Scores   -           Academic score (out of 100)
  \Extra_Curricular_Hours -   Time spent on extracurricular activities

## 📊 Dashboard & Visualizations

The dashboard includes the following components:

-   Average test scores by screen time category\

### 📉 Charts

These visualizations help identify patterns and correlations in student

------------------------------------------------------------------------

## 🔎 Key Insights

-   📈 Students with **moderate screen time (4--6 hours)** achieved the
    highest average test scores.\
-   ⚖ Balanced screen usage does not necessarily harm academic
    performance.\
-   ⚽ Students with moderate screen time also show **higher
    extracurricular participation**.\
-   👩‍🎓 Students aged **16 recorded the highest study hours and overall
    performance**.\
-   📚 Academic success appears more related to **balanced lifestyle
    habits and study time** rather than minimal screen usage.
