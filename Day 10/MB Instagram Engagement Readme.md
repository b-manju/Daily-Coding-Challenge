# 📊 Instagram Engagement Analysis Dashboard

## 📌 Project Overview
This project analyzes Instagram post performance data to uncover insights about audience engagement, content effectiveness, and growth trends using Power BI.

## 🎯 Objectives
- Analyze engagement metrics across post types
- Identify trends in reach and impressions
- Evaluate impact of captions and hashtags
- Understand follower vs engagement relationship

## 📂 Dataset Features
- Post_ID
- Post_Date
- Post_Type (Image, Video, Reel, Story)
- Caption_Length
- Hashtags_Used
- Likes, Comments, Shares, Saves
- Reach, Impressions
- Follower_Count

## ⚙️ Steps Performed

### 🧹 Data Cleaning
- Removed duplicates
- Handled missing/null values
- Corrected data types

### 🔄 Data Transformation
- Created Month-Year column
- Prepared data for trend analysis

### 📐 DAX Measures
- Engagement Rate  
- Interaction Ratio  
- Total Engagement  

### 📊 Dashboard Features
- KPI Cards (Total Reach, Impressions, Engagement Rate, Total Posts)
- Trend Analysis (Reach, Impressions, Engagement over time)
- Post Type Comparison
- Top 5 Performing Posts
- Scatter Plot (Follower Count vs Engagement)
- Slicers (Post Type, Month-Year)

## 📊 Key Insights
- Reels generate the highest engagement rate
- Moderate hashtag usage gives better performance
- Engagement is not directly proportional to follower count
- Certain months show peak reach and impressions

## 🛠 Tools Used
- Power BI
- Power Query
- DAX
